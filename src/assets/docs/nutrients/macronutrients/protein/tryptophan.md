# Tryptophan
Tryptophan is an essential amino acid.

## Daily recommendations
- Men: 5mg/kg/day;
- Women: 5mg/kg/day;
- Pregnancy: 7mg/kg/day;
- Lactation: 9mg/kg/day;

## Food sources
- Egg;
- Seaweed;
- Fish;
- Soy;
- Seeds;
- Sofu.
