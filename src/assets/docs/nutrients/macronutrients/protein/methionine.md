# Methionine
Methionine is an essential amino acid.

## Daily recommendations
- Men: 19mg/kg/day;
- Women: 19mg/kg/day;
- Pregnancy: 25mg/kg/day;
- Lactation: 26mg/kg/day;

## Food sources
- Egg;
- Fish;
- Seaweed;
- Soy;
- Seeds;
- Nuts;
- Cheese;
- Lamb;
- Beef;
- Pork.
