# Phenylalanine
Phenylalanine is an essential amino acid.

## Daily recommendations
- Men: 33mg/kg/day;
- Women: 33mg/kg/day;
- Pregnancy: 44mg/kg/day;
- Lactation: 51mg/kg/day;

## Food sources
- Egg;
- Seaweed;
- Fish;
- Soy;
- Seeds;
- Dairy;
- Tofu.
