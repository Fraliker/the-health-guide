# Lysine
Lysine is an essential amino acid.

## Daily recommendations
- Men: 38mg/kg/day;
- Women: 38mg/kg/day;
- Pregnancy: 51mg/kg/day;
- Lactation: 52mg/kg/day;

## Food sources
- Egg white;
- Soy;
- Cheese;
- Beef;
- Pork.
